What we have done so far

Flood Watch Device

This is a Flood Watch Device. This device would alert people who live, farm, or have businesses along these bodies of water of potential flood dangers. It would do this by continuously monitoring water depth near these streams and transmitting these data to areceiving station inside the residence or business. When the water reaches a significant depth, analarm would sound. Both the sensor station and the receiving station work on battery power.

Automaited Hand

This is an automated hand prototype. This prototype of five-fingered hand, simulate the motion of a real human hand. the prototype is made using Intel Edison and python as a programing language.

Facial Recognition Lock System

This is a Facial Recognition Lock System.This prototype device used Intel Edison, web cam , servos and python as a programming language. The prototype has capable of capturing images, detecting faces and, comparing the face with a facial library of authorized users, and activating the servo to open the door lock if the user is recognized or ensuring the door remains locked if the user is not authorized.
